# EditMot (OSS Name : MameBake3D)
モーキャプ変換物理編集リターゲット
モーションキャプチャをブラシで盛るように編集 fbxにしてUnityへ
(bvh, fbx, Unity, Maya, VRoid対応)
FBX Animation Tool.(Load Baked FBX File. Edit Animation. Setting bullet params. Output to FBX file.)
